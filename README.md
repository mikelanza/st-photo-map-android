## Installation

1. Add the JitPack repository to your root build.gradle at the end of repositories:
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
2. Add the dependency:
```
dependencies {
    implementation 'com.github.mikelanza:st-photo-map-android:latest-release'
}
```
